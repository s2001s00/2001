# 🎮 咖啡馆经营 - 手机游戏

一个基于Web的移动端经营模拟游戏，支持应用内充值变现。

## 🌟 游戏特性

- ☕ **经营循环**：生产、销售咖啡赚取金币
- 💰 **应用内充值**：玩家可充值金币加速游戏进程
- ⬆️ **升级系统**：升级咖啡机、提升收入
- 💾 **自动存档**：游戏进度自动保存到本地
- 📱 **移动适配**：完美支持各种移动设备
- 🎯 **简单易上手**：全程点击操作

## 🎮 核心游戏机制

### 收入循环
1. **顾客生成** - 每3秒自动生成新顾客
2. **咖啡生产** - 每2秒自动生产咖啡
3. **服务顾客** - 点击"服务顾客"按钮赚取金币
4. **升级机器** - 花费金币升级咖啡机，提升收入

### 升级收益
- 每升级一级咖啡机，单位收入提升50%
- 升级成本按1.5倍递增
- 达成顾客数量目标可升级等级

## 📊 数据持久化

- 游戏状态自动保存到浏览器LocalStorage
- 每30秒自动保存一次
- 页面关闭前自动保存
- 下次打开时自动恢复进度
